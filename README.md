# AI Interviewer Bot

Production-ready starter for an **AI Interviewer Bot** with:
- FastAPI backend (question generation, answer evaluation, feedback)
- React + Vite frontend (interview flow UI)
- MongoDB persistence
- OpenAI integration (LLM + Whisper STT)

See:
- `docs/architecture.md` for architecture, folder structure, and deployment.
- `backend/README.md` for backend setup.
- `frontend/README.md` for frontend setup.
